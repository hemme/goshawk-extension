# SGF/HEN from Image

This is a semiautomatic converter from diagram images of the game Go (Weiqi, Baduk) to SGF/HEN format.

This tool is a fork of the one originally developed by [Kaorahi](https://github.com/kaorahi/lizgoban/tree/main/src/sgf_from_image) - try an [online demo](http://kaorahi.github.io/lizgoban/src/sgf_from_image/sgf_from_image.html).
